# Prompt: Tech Radar — Software, AI & Homelab

## Objetivo

Generar un newsletter técnico, fácil de leer y útil para developers full-stack e IT.

El contenido debe cubrir tendencias de ingeniería de software, no solo frontend ni solo un vendor.

## Formato de salida

Crear dos versiones:

1. Texto plano de respaldo.
2. HTML mobile-first compatible con clientes de correo.

## Reglas de diversidad

- Usar entre 6 y 8 temas reales.
- Cubrir al menos 5 áreas distintas.
- Evitar que una sola empresa, framework o dominio domine la edición.
- Máximo 1 tema por vendor principal, salvo excepción justificada.
- Usar fuentes originales o cercanas a la fuente original.

## Áreas sugeridas

1. Frontend, JavaScript o TypeScript.
2. CSS, diseño UI, UX engineering, accesibilidad o web platform.
3. Backend, runtime, bases de datos o arquitectura de software.
4. DevOps, platform engineering, CI/CD, testing u observabilidad.
5. IA aplicada a desarrollo, agentes, modelos, tooling o evaluación.
6. Homelab, self-hosting, Linux, networking, contenedores o automatización personal.
7. Arquitectura de software: distributed systems, event-driven, resiliencia, performance, API design o cloud architecture.
8. Repo Radar: repositorio open source relevante con release o actividad reciente.

## Fuentes recomendadas

- W3C, WHATWG, TC39, MDN, Chrome Developers, WebKit, Mozilla Hacks, web.dev.
- TypeScript Blog, React Blog, Node.js Blog, Vite, Vitest, Vue, Svelte, Angular, Astro, Bun, Deno.
- PostgreSQL, SQLite, Redis, Valkey, Kafka, Kubernetes, CNCF, OpenTelemetry.
- Harness, GitHub Changelog, GitLab, Docker, Terraform, OpenTofu, Pulumi, Grafana, Prometheus, Datadog.
- OpenAI, Anthropic, Google DeepMind, Meta AI, Mistral, Hugging Face, LangChain, LlamaIndex, Ollama, LiteLLM, vLLM.
- Proxmox, TrueNAS, Home Assistant, Tailscale, Docker, Portainer, Uptime Kuma, Immich, Jellyfin, Nextcloud, Pi-hole, AdGuard Home, OpenWrt, pfSense, OPNsense, k3s, Longhorn.

## Secciones obligatorias

- Resumen editorial.
- Esta edición, en breve.
- Tarjetas de temas.
- Qué vale la pena probar esta semana.
- Lectura rápida.
- Fuentes de esta edición.

## Validación final

- No hay placeholders visibles.
- Todos los links apuntan a fuentes originales.
- Hay entre 6 y 8 tarjetas.
- Hay al menos 5 áreas cubiertas.
- Ningún vendor domina la edición.
- Existe una versión de texto plano y una versión HTML.
